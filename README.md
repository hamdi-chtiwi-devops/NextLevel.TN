# NextLevel.TN - A Modern, AI-Powered E-Learning Platform

Welcome to NextLevel.TN, a fully-featured, intelligent, and modern e-learning platform designed to help users master new skills in technology. Built with a powerful stack including Next.js, Firebase, and Google's Genkit, this platform provides a seamless and interactive learning experience.

## âœ¨ Key Features

### 1. User Authentication & Profile Management
- **Secure Sign-Up/Sign-In**: Users can create an account or sign in using their email and password or with a single click via Google authentication.
- **Personalized Profiles**: A dedicated profile page where users can update their personal information, manage their password, and view their progress.
- **Auth-Aware Navigation**: The application provides a separate landing page for new visitors and a protected, feature-rich experience for authenticated users.

### 2. Core Learning Experience
- **Curated Course Catalog**: A comprehensive list of courses covering various topics like Web Development, UI/UX, Data Science, and more.
- **Detailed Course Pages**: Each course has a dedicated page with a detailed description, instructor information, and course content broken down into lessons.
- **Engaging Learning Paths**: Expert-curated roadmaps that guide students through a sequence of courses to master a specific skill or technology domain.
- **Hands-On Projects**: A dedicated projects section with over 10 step-by-step, real-world projects to help students apply what they've learned.

### 3. AI-Powered Learning Tools (Powered by Genkit)
- **AI Coding Arena**: An interactive space where users can request coding challenges by topic and difficulty, write code in an in-browser editor, and receive instant, AI-driven feedback and evaluation.
- **AI-Generated Quizzes**: Both within courses and on the dashboard, users can generate quizzes on any topic to test their knowledge.
- **AI Question & Answer**: An intelligent assistant on the dashboard that can answer any question a student might have, from complex technical concepts to simple definitions.
- **Personalized Recommendations**: The dashboard features an AI-powered recommendation engine that suggests relevant courses based on a user's learning history and preferences.

### 4. Rich, Interactive UI/UX
- **Modern & Responsive Design**: Built with ShadCN UI components and Tailwind CSS, the interface is clean, modern, and fully responsive across all devices.
- **Light & Dark Mode**: Users can switch between light and dark themes for a comfortable viewing experience at any time.
- **Interactive Dashboard**: A central hub for learners that welcomes them back, shows their enrolled courses, and provides access to all AI-powered learning tools.
- **Certificate Generation**: Upon completing a course, users can generate and download a personalized PDF Certificate of Completion.

### 5. Monetization
- **Subscription Plans**: A professional pricing page that offers both monthly and annual subscription plans, allowing for future monetization.

##  workflow

1.  **Visitor Arrives**: A new visitor is greeted by a modern landing page, outlining the platform's key features and benefits.
2.  **Authentication**: The user can either sign up for a new account or log in. Google Sign-In provides a one-click entry point.
3.  **Dashboard**: Upon logging in, the user is taken to their personal dashboard. Here, they get a quick overview of their progress, can start an AI quiz, ask a question, or explore recommended courses.
4.  **Learning**: The user can browse the **Courses**, **Learning Paths**, or **Projects** pages to start their journey.
5.  **Practice**: To sharpen their skills, the user can head to the **AI Coding Arena** to tackle AI-generated challenges.
6.  **Profile & Settings**: The user can manage their profile information, change their theme, and view billing options at any time from the user menu.

This project showcases a modern approach to building a full-stack, AI-integrated web application.
