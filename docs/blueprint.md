# **App Name**: AcademiAI

## Core Features:

- User Authentication: Secure user authentication using Firebase Auth with role-based access control (Admin, Instructor, Student).
- Course Management: Create, read, update, and delete (CRUD) operations for courses, including details like title, description, category, instructor, price, and media.
- Lesson and Quiz Creation: Enable instructors to create lessons with text/video content and auto-generate quizzes using Firebase AI.
- AI-Driven Recommendations: Personalized course recommendations for students based on their learning history and preferences using Firebase AI tool.
- Progress Tracking: Real-time tracking of student progress, quiz scores, and course completion status.
- Community Forum: Real-time discussion threads per course for students and instructors to interact, ask questions, and share knowledge.
- User Profile Management: Profile editing and avatar uploads via Firebase Storage to enable personalized profiles for all roles.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke trust and knowledge.
- Background color: Very light blue (#F0F2F9) for a clean, calming learning environment.
- Accent color: Purple (#7E57C2) to highlight important elements and CTAs.
- Body font: 'PT Sans', sans-serif, providing a modern and readable feel for all content.
- Headline font: 'Space Grotesk', sans-serif, offering a contemporary and techy touch for titles and headings.
- Consistent and clear icons from a library like FontAwesome or Material Icons, aligned with the platform's theme.
- Subtle transitions and animations to enhance user experience without being distracting.
